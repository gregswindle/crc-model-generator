NullCrcClass

eslint-plugin-crc/crc~ NullCrcClass

new NullCrcClass(paramsopt)

Source:
  • crc/null-crc-class.js, line 48

Create a class representation for use in a CrcClass (Class-Responsibility-Collaboration model).

Example
const crcClass = new CrcClass({
  code: sourceCode,
  meta: {
    context,
    description,
    filePath,
    references
  },
  name: "Bravo",
  superClass: Alpha
});
