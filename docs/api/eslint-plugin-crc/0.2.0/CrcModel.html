CrcModel

new CrcModel(params)

Source:
  • crc/crc-model.js, line 32

Create a shadow Class-Responsibility-Collaboration model.

Example
const crcModel = new CrcModel({
  class,
  responsibilities,
  collaborators
});
Parameters:
Name Type Description
params Object

A parameter object that optionally sets all of the CrcModel's properties.

Properties
Name Type Attributes Description
class CrcClass <optional>

A class representation.
collaborators array.<CrcClass> <optional>

A collection of zero or more CrcClasses that this object depends on.
responsibilities array.<CrcResponsibility> <optional>

A list of data this object must maintain and/or operations it must perform.
